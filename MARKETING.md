# Her Desk — 企业级 Java 远程桌面解决方案

> **一个 JAR，双端齐备。内网极速直连，公网一键穿透。**
> 无需浏览器、无需复杂配置，打开就能远程操控另一台电脑。

---

**Her Desk 把这件事做到了极致简单：**

| 对比项 | 常见远程方案 | Her Desk |

## 三大核心能力

### ① 内网直连 —— 局域网零延迟操控

同一 Wi-Fi / 同一办公室网络，**即连即用**。

- 被控端一键「启动服务」，自动显示本机 IP
- 控制端填入 IP + 端口（默认 5900），点击「连接」
- 实时显示远程桌面，**完整键鼠控制**（移动、点击、滚轮、组合键）
- 底部实时 **帧率监控**，性能状态心中有数

**典型场景：**
- IT 运维人员远程维护办公室电脑
- 设计师在家操控公司高配置工作站
- 技术支持人员远程协助同事排障

![内网直连控制端](docs/screenshots/client-direct-connected.png)

---

### ② 公网中继 —— 突破内网，随时随地连回家

不在同一网络？没关系。**不需要路由器端口映射，不需要买穿透服务。**

- 在公网 VPS 部署中继（一个 JAR，一行命令）
- 被控端选「公网中继」，**自动生成随机房间号**，设置房间密码
- 控制端填相同中继地址 + 房间号 + 密码，即可连接
- 密码错误直接拒绝，防止陌生人误入

**典型场景：**
- 出差途中远程访问家里/公司电脑
- 客户现场电脑需要总部工程师远程支援
- 多地点分支机构统一远程运维

![公网中继被控端](docs/screenshots/server-relay-mode.png)

---

### ③ 沉浸式全屏 —— 像坐在电脑前一样操作

连接成功后，一键「全屏」：

- 远程画面铺满整个屏幕，**无边框沉浸体验**
- 右上角「退出全屏」按钮，**Esc 键**也可快速退出
- 全屏下键鼠操作与窗口模式完全一致
- 多次进出全屏稳定可靠，资源自动回收

**典型场景：**
- 远程演示、远程培训
- 全屏操作设计软件、IDE、浏览器
- 需要大屏沉浸感的任何远程办公场景

![全屏远程桌面](docs/screenshots/fullscreen-mode.png)

## 技术亮点 —— 看不见的牛逼，才是真的牛逼

### 智能画面引擎

不是粗暴的「每秒截一张全屏图」，而是：

- **首帧全屏传输**，后续 **96×96 区块增量差分**，只传变化区域
- **三档画质可调**：高清晰 / 均衡 / 流畅，网络差也能用
- **静止自动降频，操作自动提速**，最高约 50 FPS
- 支持 **Retina 高分屏**自动缩放，坐标精准换算

> 实测：1512×982 分辨率 Mac 桌面，均衡画质下流畅操控浏览器、IDE 无压力。

### 自研 HDRD 二进制协议

- TCP 长连接，稳定可靠
- 握手阶段交换屏幕几何信息，多分辨率自适应
- 键鼠事件独立通道，操作响应跟手
- 心跳保活 + 断线自动检测

### 全程可观测

被控端、控制端均内置 **日志面板**：

- 连接每一步都有记录（注册、加入、握手、首帧）
- 网络错误 **中文详细提示**（超时、密码错误、房间未找到……）
- 画质切换、全屏进出、断开原因全程留痕
- **出了问题不用猜，看日志就知道**

![控制端日志与状态](docs/screenshots/client-relay-mode.png)

### 轻量部署，极简运维

```
一个 JAR = 被控端 + 控制端 + 启动器
一个中继 JAR = 公网穿透服务
```

- JDK 8 ~ 21 均可运行，**Windows 10/11、macOS 10.15+**
- 提供 Windows `.bat` 和 macOS `.command` 双击启动脚本
- 无需数据库、无需 Redis、无需 Docker
- VPS 上一行命令启动中继：`java -jar her-desk-relay.jar 9000`

- 远程画面区域 + 实时帧率 + 状态栏
- 连接 / 断开 / 全屏 三个核心按钮
- 浅蓝专业主题 UI，清爽不压抑

## 三步上手，五分钟远程

### 内网场景（最快）

```
被控端：启动 → 点「启动服务」
控制端：填 IP → 点「连接」
完成。开始远程操作。
```

### 跨网场景（同样简单）

```
VPS：java -jar her-desk-relay.jar 9000
被控端：选中继模式 → 启动服务 → 告诉对方房间号
控制端：填中继地址 + 房间号 + 密码 → 连接
完成。千里之外，如在眼前。
```
