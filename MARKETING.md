## 三大核心能力

### ① 内网直连 —— 局域网零延迟操控

同一 Wi-Fi / 同一办公室网络，**即连即用**。

- 被控端一键「启动服务」，自动显示本机 IP
- 控制端填入 IP + 端口，点击「连接」
- 实时显示远程桌面，**完整键鼠控制**（移动、点击、滚轮、组合键）
- 底部实时 **画面流畅度监控**，性能状态心中有数

**典型场景：**
- IT 运维人员远程维护办公室电脑
- 设计师在家操控公司高配置工作站
- 技术支持人员远程协助同事排障

![内网直连控制端](docs/screenshots/client-direct-connected.png)

---

### ② 公网中继 —— 突破内网，随时随地连回家

不在同一网络？没关系。**不需要路由器端口映射，不需要另买穿透服务。**

- 在云服务器上部署中继服务，**一次配置，长期使用**
- 被控端选「公网中继」，**自动生成随机房间号**，设置房间密码
- 控制端填相同中继地址 + 房间号 + 密码，即可连接
- 密码错误直接拒绝，防止陌生人误入

**典型场景：**
- 出差途中远程访问家里/公司电脑
- 客户现场电脑需要总部工程师远程支援
- 多地点分支机构统一远程运维

![公网中继被控端](docs/screenshots/server-relay-mode.png)

---

### ③ 沉浸式全屏 —— 像坐在电脑前一样操作

连接成功后，一键「全屏」：

- 远程画面铺满整个屏幕，**无边框沉浸体验**
- 右上角「退出全屏」按钮，**Esc 键**也可快速退出
- 全屏下键鼠操作与窗口模式完全一致
- 多次进出全屏稳定可靠，运行流畅

**典型场景：**
- 远程演示、远程培训
- 全屏操作设计软件、浏览器、办公软件
- 需要大屏沉浸感的任何远程办公场景

![全屏远程桌面](docs/screenshots/fullscreen-mode.png)

## 技术亮点 —— 看不见的牛逼，才是真的牛逼

### 智能画面引擎

不是粗暴的「每秒截一张全屏图」，而是：

- **首次完整呈现画面**，后续**只刷新有变化的部分**，省流量、更流畅
- **三档画质可调**：高清晰 / 均衡 / 流畅，网络差也能用
- **画面静止时自动省电省流，操作时再自动提速**
- 支持 **高清屏、高分屏**自动适配，鼠标位置精准跟手

> 实测：高分辨率 Mac 桌面，均衡画质下流畅操控浏览器、办公软件无压力。

### 自研传输协议

- 长连接保持稳定，断线自动感知
- 自动适配不同分辨率屏幕
- 键鼠操作独立传输，响应跟手
- 心跳检测，连接状态实时掌握

### 全程可观测

被控端、控制端均内置 **日志面板**：

- 连接每一步都有记录（注册、加入、连接成功、首帧画面）
- 网络错误 **中文详细提示**（超时、密码错误、房间未找到……）
- 画质切换、全屏进出、断开原因全程留痕
- **出了问题不用猜，看日志就知道**

![控制端日志与状态](docs/screenshots/client-relay-mode.png)

### 轻量部署，极简运维

```
一套主程序 = 被控端 + 控制端 + 启动器
一套中继程序 = 公网穿透服务
```

- 支持 **Windows 10/11、macOS 10.15+**
- 提供 Windows、Mac **双击启动脚本**，也可打包为 **安装包** 分发
- 无需数据库、无需额外中间件
- 云服务器上一键启动中继服务

## 三步上手，五分钟远程

### 内网场景（最快）

```
被控端：打开软件 → 点「启动服务」
控制端：填 IP → 点「连接」
完成。开始远程操作。
```

### 跨网场景（同样简单）

```
云服务器：启动中继服务
被控端：选中继模式 → 启动服务 → 告诉对方房间号
控制端：填中继地址 + 房间号 + 密码 → 连接
完成。千里之外，如在眼前。
```
